### term_xmr.py
---------------

Python3 script to display on a terminal the live price of a cryptocurrency, Monero in this case, in US Dollar, Bitcoin, Euro.

![](gif_xmr.gif)

### pf_xmr.py
-------------
(21/04/22)
This script allows to check your XMR portfolio in $, ₿, €. Some major coins are added.
